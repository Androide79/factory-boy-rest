Welcome to Factory Boy Rest's documentation!
============================================

A REST test fixtures replacement for Python.

This is intented to be used on functional tests, to create REST-based entities
through the same `Factories`_ interface provided by `Factory Boy`_

.. _Factories: http://factoryboy.readthedocs.org/en/latest/reference.html#the-factory-class
.. _`Factory Boy`: https://github.com/rbarrois/factory_boy
